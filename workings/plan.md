## <u>Good Vibrations</u>
* Mixed Reality Game that allows the user to control the game by using voice commands, keyboard, and mouse.
* Main Character - Dog

### World
  * Park
  * Trees, Grass, flowers, Bushes
  * Dog & Human NPCs
  * Bone Collectables 

## Speech rules
###### 1. Start Menu
 * Start/Play/Continue/Begin
 * Exit/Quit


###### 2. Select Dog Menu - 6 Dogs
 * Next/No/Nah/Nope
 * Select/Yes/Yeah


###### 3. In-Game Menu
  * Pause/Hold
  * Resume/Back
  * Main Menu


###### 4. Idle Movement
  * Idle/Yield/Stop/Halt


###### 5. Walk Movement
  * Walk/Go/Forward/Stroll/Wander


###### 6. Run Movement
  * Run/Jog/Dash
